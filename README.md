# How to launch
## Prerequisite
 - ruby
 - bundler
## Initial setting
  - download the files from github to local
  - make `*.csv` file
  - set path to `*.csv` on `DATA_FILE` in `const.rb`
## Install gems
  ```
  bundle install
  ```
 ## Launch `main.rb`
 ```
 bundle exec ruby main.rb
 ```
 ## Access to app
  - http://localhost:4567
